# LouisvilleTeenCrimes
Tracking crime trends among Louisville youth from 2020 to 2026 using LMPD data 


Because this data deals with teens ages 13 to 17, the law protects their identity leading to certain columns/information in my open records request was redacted. Those columns will be the first lines of code I will drop. 

Here's a list of the column names with "redacted" information: 
- arrest_number 
- defendants.person_id
- defendants.master_person_id
